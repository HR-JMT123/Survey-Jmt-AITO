1. Prompt Engineering & Generative AI
(1 = น้อยที่สุด, 5 = มากที่สุด)
1.1 ความเข้าใจหลักการพื้นฐานของ Prompt Engineering เพิ่มขึ้นระดับใด
1.2 ความเข้าใจในการประยุกต์ใช้งาน Chat GPT เพื่อช่วยงานธุรกิจเพิ่มขึ้นระดับใด
1.3 ความเข้าใจในการประยุกต์ใช้งาน Google Gemini เพิ่มขึ้นระดับใด
1.4 ความสามารถในการเขียนคำสั่ง (Prompt) ที่ซับซ้อนและมีเงื่อนไขเพิ่มขึ้นระดับใด
1.5 แนวทางในการใช้ AI ช่วยระดมสมอง (Brainstorming) และวางแผนงานชัดเจนขึ้นระดับใด
1.6 ความสามารถในการใช้ AI ช่วยสรุปเนื้อหาเอกสารหรือข้อมูลเพิ่มขึ้นระดับใด
1.7 ความเข้าใจข้อจำกัดและการตรวจสอบความถูกต้องของ AI (AI Hallucination) เพิ่มขึ้นระดับใด
1.8 ความเข้าใจด้านความปลอดภัยของข้อมูล (Data Privacy) ในการใช้ AI เพิ่มขึ้นระดับใด
2. n8n & Workflow Automation
2.1 ความเข้าใจในการใช้งาน n8n และสร้าง Workflow x Spreadsheet เพิ่มขึ้นระดับใด
2.2 ความเข้าใจหลักการทำงานของระบบอัตโนมัติ (Automation) เพิ่มขึ้นระดับใด
2.3 ความเข้าใจการเชื่อมต่อ API พื้นฐานระหว่างแอปพลิเคชันเพิ่มขึ้นระดับใด
2.4 ความเข้าใจการตั้งค่าเงื่อนไข (Conditions/If-Else) ใน Workflow เพิ่มขึ้นระดับใด
2.5 แนวทางการประยุกต์ใช้ n8n ร่วมกับ Google Sheets เพื่อลดงานซ้ำซ้อนชัดเจนขึ้นระดับใด
2.6 ความเข้าใจการตั้งค่า Trigger เพื่อให้ Workflow ทำงานอัตโนมัติเพิ่มขึ้นระดับใด
3. ความสามารถหลักของ Claude 7 ข้อ
3.1 การใช้ Claude ช่วยสรุปเนื้อหา (Summarization) จะช่วยงานคุณได้ระดับใด
3.2 การใช้ Claude ช่วยเขียนและปรับแต่งข้อความ (Writing) จะช่วยงานคุณได้เพิ่มขึ้นระดับใด
3.3 การใช้ Claude ช่วยวิเคราะห์ข้อมูลและสร้างตาราง (Data Analysis) จะช่วยงานคุณได้เพิ่มขึ้นระดับใด
3.4 การใช้ Claude ช่วยตอบคำถามและสกัดข้อมูล (Q&A & Extraction) จะช่วยงานคุณได้เพิ่มขึ้นระดับใด
3.5 การใช้ Claude ช่วยแปลภาษา (Translation) จะช่วยงานคุณได้เพิ่มขึ้นระดับใด
3.6 การใช้ Claude ช่วยระดมสมองและวางแผน (Brainstorming & Planning) จะช่วยงานคุณได้เพิ่มขึ้นระดับใด
3.7 การใช้ Claude ช่วยวิเคราะห์รูปภาพและไฟล์เอกสาร (Vision & Files) จะช่วยงานคุณได้เพิ่มขึ้นระดับใด
